# Feihong's Nim Quickstart

## Installation

    curl https://nim-lang.org/choosenim/init.sh -sSf | sh
    choosenim stable

Then add `export PATH=~/.nimble/bin:$PATH` to `~/.bash_profile`.

Source: https://nim-lang.org/install_unix.html

## Commands

Build a program

    nim c hello.nim

Build and run a program

    nim c -r hello.nim
